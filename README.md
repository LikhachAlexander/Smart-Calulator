# Smart-Calulator
Smart calculator project from JetBrainsAcademy Python course
# Features
* variables
* functions
* access to previous result
* commands(start with '/')
# How to run
* Run "calculator.py" script
```
python calculator.py
```
* Run cmd application in dist folder
# Example
```
>a = 2
>b = 2 * 3
>c = a * b
>/variables
variables
    "PI": 3.141592653589793
    "e": 2.718281828459045
    "a": 2.0
    "b": 6.0
    "c": 12.0
>/functions
functions
    fact
    exp
    sin
    cos
    log
>fact(c)
479001600.0
>a + 2 * b + c * (a + b)
110.0
```
